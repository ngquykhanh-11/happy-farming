# Happy Farming Shop 

This is a class project me and my group created last year. Happy Farming is a dynamic web application designed to bridge the gap between farmers and consumers using Blockchain technology. 
The platform ensures transparency in food origin and provides a seamless trading experience through a modern, interactive interface.

- Frontend: ReactJS, TailwindCSS
- Backend: Backend: Python (FastAPI).
- Database: SQL (using with )
- Blockchain: Solidity (Smart Contracts), RemixIDE
- Tools: Metamask, Postman (API Testing).
